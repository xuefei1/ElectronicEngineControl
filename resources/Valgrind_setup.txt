Valgrind setup

1. cd into the source directory.

2. Run ./autogen.sh to setup the environment(if not working, check **)

3. Run ./configure, with some options if you wish.  The only interesting
    one is the usual --prefix=/where/you/want/it/installed.

4. Run "make".

5. Run "sudo make install", possibly as root if the destination permissions require that.

6. See if it works.  Try "valgrind ls -l".
